
Script em Python para checagem básica de conectividade e SNMP em múltiplos hosts.
Realiza testes de internet, ping e SNMPWALK de forma interativa via terminal.

Funcionalidades

Teste de conectividade com a internet via DNS Google.

Entrada interativa de múltiplos IPs.

Teste de ping individual para cada host.

Entrada de community SNMP por host.

Suporte a porta SNMP personalizada ou padrão 161.

Execução de snmpwalk para validação SNMP.

Exibição clara dos resultados finais.

Requisitos

Python 3.x

Sistema Linux ou Unix-like

Pacotes instalados:

ping

snmpwalk (pacote snmp ou net-snmp)
